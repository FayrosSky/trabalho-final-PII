## 📝 Breve Descrição do Portfólio

O **CodeAprende** é um portfólio educacional desenvolvido com tecnologias web básicas (HTML, CSS e JavaScript) que serve como plataforma de ensino para quem quer aprender desenvolvimento front-end do zero.

## 🚧 Principais Desafios Encontrados

### 1. **Criação de Layout Responsivo**
- **Desafio:** Fazer o site funcionar bem em diferentes tamanhos de tela (mobile, tablet, desktop)
- **Solução:** Implementação de media queries e Flexbox para layouts fluidos

### 2. **Organização do Código**
- **Desafio:** Manter o código limpo e organizado em múltiplas páginas (13 páginas no total)
- **Solução:** Uso de CSS global reutilizável e estrutura de pastas bem definida

### 3. **Símbolos Especiais e Codificação**
- **Desafio:** Caracteres especiais (acentos, emojis) não apareciam corretamente
- **Solução:** Implementação de `<meta charset="UTF-8">` em todas as páginas

### 4. **Sistema de Navegação**
- **Desafio:** Criar menu de lições lateral que mostra progresso e permite navegação rápida
- **Solução:** Lista ordenada `.lessons-list` com classes para página ativa

### 5. **Destaque de Código**
- **Desafio:** Mostrar trechos de código de forma clara e colorida
- **Solução:** Criação da classe `.code-example` com estilos específicos para tags, atributos, strings e comentários

### 6. **Manutenção de Consistência**
- **Desafio:** Manter o mesmo design em todas as páginas do curso
- **Solução:** CSS global em `style.css` com classes reutilizáveis (`.container`, `.btn`, `.course-header`)
