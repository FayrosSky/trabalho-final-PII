📖 O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem que define a estrutura de páginas web [web:1].

💡 Analogia Simples

Imagine que um site é como uma casa:

  • HTML = Estrutura (alicerces, paredes, tetos)
  • CSS = Decoração (pintura, móveis, estilo)
  • JavaScript = Funcionalidades (eletricidade, água, portão automático)

🏗️ Estrutura Básica de um Arquivo HTML

Todo arquivo HTML segue esta estrutura:

exemplo-basico.html 
<!-- 1. Declara que é HTML5 -->
<!DOCTYPE html>

<!-- 2. Elemento raiz da página -->
<html lang="pt-BR">

<!-- 3. Cabeçalho (metadados) -->
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Primeira Página</title>
</head>

<!-- 4. Corpo (conteúdo visível) -->
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro site!</p>
</body>

</html>

📌 Explicação linha por linha:

<!DOCTYPE html> Informa ao navegador que é HTML5 [web:1]
<html lang="pt-BR"> Elemento raiz, define o idioma como português do Brasil
<head> Cabeçalho: contém metadados (informações sobre a página)
<meta charset="UTF-8"> Codificação de caracteres (permite acentos e emojis)
<title> Título que aparece na aba do navegador
<body> Corpo: conteúdo visível na página

🛠️ Ferramentas que Você Precisa

Para começar a programar em HTML, você precisa apenas de:

📝

1. Editor de Código

Recomendado: VS Code (gratuito)

Otimize sua programação com 자동 완성, syntax highlighting e extensões.

🌐

2. Navegador

Recomendados: Chrome, Firefox ou Edge

Use o DevTools (F12) para inspecionar e debugar código.

✍️ Exercício Prático: Sua Primeira Página

📝 Criando seu Primeiro Site

  1. Abra o VS Code
  2. Crie uma nova pasta chamada meu-primeiro-site
  3. Dentro dela, crie um arquivo chamado index.html
  4. Cole o código de exemplo acima
  5. Salve o arquivo (Ctrl + S)
  6. Abra o arquivo no navegador (duplo clique ou arraste)
  7. Parabéns! 🎉 Você criou seu primeiro site!

🎯 Resumo da Aula

  • HTML = HyperText Markup Language
  • HTML define a estrutura do conteúdo
  • Toda página HTML tem: <!DOCTYPE>, <html>, <head>, <body>
  • Você precisa apenas de VS Code + Navegador para começar
  • HTML é a base de todo desenvolvimento web
