Formulários HTML5

📖 O que são Formulários?

Formulários são usados para coletar dados do usuário: login, cadastro, contato, pesquisa e muito mais [web:1].

formulario-basico.html 
<form action="/enviar.php" method="POST">
    
    <!-- Campos do formulário -->
    <input type="text" name="nome">
    <input type="email" name="email">
    <buttonEnviar</button>
    
</form>

🎯 1. A Tag <form>

A tag <form> envolve todos os campos do formulário:

Atributo Descrição
action URL para onde os dados serão enviados
method GET (URL) ou POST (corpo da requisição)

📌 GET vs POST

  • GET: Dados visíveis na URL (não use para senhas!)
  • POST: Dados ocultos no corpo da requisição (mais seguro)

✏️ 2. Input - O Campo Mais Versátil

O <input> tem vários tipos diferentes:

Tipos de Input:

tipos-input.html 
<!-- Texto comum -->
<input type="text" placeholder="Seu nome">

<!-- E-mail (valida automaticamente) -->
<input type="email" placeholder="email@exemplo.com">

<!-- Senha (esconde caracteres) -->
<input type="password" placeholder="******">

<!-- Número -->
<input type="number" min="0" max="100">

<!-- Botão de rádio (seleção única) -->
<input type="radio" name="genero"> Masculino
<input type="radio" name="genero"> Feminino

<!-- Checkbox (múltiplas opções) -->
<input type="checkbox"> Aceito os termos

<!-- Data -->
<input type="date">

<!-- Telefone -->
<input type="tel" placeholder="(11) 99999-9999">

<!-- Botão de envio -->
<input type="submit" value="Enviar">

🏷️ 3. Atributos Importantes do Input

atributos-input.html 
<input 
    type="text"
    name="nome"           <!-- Nome do campo (enviado ao servidor) -->
    id="nome"             <!-- ID único (para label) -->
    placeholder="Digite seu nome"  <!-- Texto de exemplo -->
    required              <!-- Campo obrigatório -->
    maxlength="50"        <!-- Máximo de caracteres -->
    value="valor inicial" <!-- Valor padrão -->
>

📋 4. Label - Rótulo do Campo

O <label> dá um nome ao campo (importante para acessibilidade):

label.html 
<!-- Melhor prática: vincular com 'for' e 'id' -->
<label for="email">E-mail:</label>
<input type="email" id="email" name="email">

<!-- Alternativa: envolver o input -->
<label>
    Nome:
    <input type="text" name="nome">
</label>

💬 5. Textarea - Campo de Texto Múltiplas Linhas

textarea.html 
<label for="mensagem">Mensagem:</label>
<textarea 
    id="mensagem"
    name="mensagem"
    rows="5"           <!-- Número de linhas -->
    placeholder="Digite sua mensagem..."
    required
></textarea>

📝 6. Select - Menu Suspenso (Dropdown)

select.html 
<label for="cidade">Cidade:</label>
<select id="cidade" name="cidade">
    <option>Selecione uma cidade</option>
    <option value="sp">São Paulo</option>
    <option value="rj">Rio de Janeiro</option>
    <option value="bh" selected>Belo Horizonte</option>
</select>

🔘 7. Button - Botões

botoes.html 
<!-- Botão de envio (padrão do form) -->
<button type="submit">Enviar</button>

<!-- Botão de reset (limpa o form) -->
<button type="reset">Limpar</button>

<!-- Botão comum (para JavaScript) -->
<button type="button">Clique Mim</button>

✅ 8. Validação de Formulário

HTML5 tem validação nativa!

validacao.html 
<!-- Campo obrigatório -->
<input type="text" required>

<!-- E-mail válido -->
<input type="email" required>

<!-- Número entre 0 e 100 -->
<input type="number" min="0" max="100">

<!-- Mínimo e máximo de caracteres -->
<input type="text" minlength="3" maxlength="20">

<!-- Expressão regular (padrão) -->
<input type="text" pattern="[A-Za-z]{3}">

<!-- Campo com valor padrão -->
<input type="text" value="valor inicial">

✍️ Exercício Prático: Formulário de Contato

📝 Criando um Formulário Completo

Crie um arquivo contato.html com:

  1. Um <form> com method="POST"
  2. Input tipo text para nome (com required)
  3. Input tipo email para e-mail (com required)
  4. Input tipo tel para telefone
  5. Textarea para mensagem (5 linhas, required)
  6. Select para assunto (3 opções)
  7. Checkbox para "Aceito receber novidades"
  8. Botão de envio

📝 Código Completo do Formulário

formulario-contato.html 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Formulário de Contato</title>
</head>
<body>
    <h1>Fale Conosco</h1>
    
    <form action="/enviar" method="POST">
        
        <!-- Nome -->
        <label for="nome">Nome:</label>
        <input 
            type="text" 
            id="nome" 
            name="nome"
            required
            placeholder="Seu nome completo"
        >
        
        <!-- E-mail -->
        <label for="email">E-mail:</label>
        <input 
            type="email" 
            id="email" 
            name="email"
            required
            placeholder="seu@email.com"
        >
        
        <!-- Telefone -->
        <label for="telefone">Telefone:</label>
        <input 
            type="tel" 
            id="telefone" 
            name="telefone"
            placeholder="(11) 99999-9999"
        >
        
        <!-- Assunto -->
        <label for="assunto">Assunto:</label>
        <select id="assunto" name="assunto"
            <option>Selecione um assunto</option>
            <option value="dúvida">Dúvida</option>
            <option value="sugestao">Sugestão</option>
            <option value="reclamac"ao>Reclamação</option>
        </select>
        
        <!-- Mensagem -->
        <label for="mensagem">Mensagem:</label>
        <textarea 
            id="mensagem" 
            name="mensagem"
            rows="5"
            required
            placeholder="Sua mensagem..."
        ></textarea>
        
        <!-- Checkbox -->
        <label>
            <input type="checkbox" name="news">
            Quero receber novidades por e-mail
        </label>
        
        <!-- Botão -->
        <button type="submit">Enviar Mensagem</button>
        
    </form>
</body>
</html>

🎯 Resumo da Aula

Tópicos Aprendidos:

  • <form> - Envolve todo formulário
  • <input> - Campo versátil (text, email, password, number, radio, checkbox...)
  • <label> - Rótulo do campo (acessibilidade)
  • <textarea> - Texto múltiplas linhas
  • <select> + <option> - Menu suspenso
  • <button> - Botões
  • Atributos: required, placeholder, min, max, pattern
  • Valores: GET vs POST
