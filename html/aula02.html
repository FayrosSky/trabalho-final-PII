Tags Básicas de Texto

📖 O que são Tags HTML?

Tags são os "comandos" do HTML. Elas dizem ao navegador como exibir o conteúdo. A maioria das tags vem em pares: <tag> e </tag> [web:1].

explicacao-tag.html 
<!-- Tag ABERTA -->
<p>
    Este é o conteúdo
<!-- Tag FECHADA (com /) -->
</p>

🎯 1. Títulos (Headings)

HTML tem 6 níveis de títulos, do <h1> (mais importante) ao <h6> (menos importante):

titulos.html 
<h1>Título Principal da Página</h1>
<h2>Subtítulo</h2>
<h3>Seção</h3>
<h4>Subseção</h4>
<h5>Detalhe</h5>
<h6>Informação menor</h6>

⚠️ Dica Importante

Use apenas um <h1> por página! Ele define o tema principal para SEO e acessibilidade.

📝 2. Parágrafos

O <p> é a tag para parágrafos de texto:

paragrafos.html 
<p>
    Este é um parágrafo de texto. 
    O navegador adiciona espaço antes e depois automaticamente.
</p>

<p>
    Este é outro parágrafo.
</p>

✏️ 3. Formatação de Texto

Negrito (Bold)

negrito.html 
<!-- Forma SEMÂNTICA (recomendada) -->
<strong>Este texto é importante</strong>

<!-- Forma visual (antiga) -->
<b>Este texto está em negrito</b>

Itálico

itálico.html 
<!-- Forma SEMÂNTICA (recomendada) -->
<em>Este texto é enfatizado</em>

<!-- Forma visual (antiga) -->
<i>Este texto está em itálico</i>

Outras tags de formatação:

<u> Texto sublinhado
<s> Texto riscado (texto deletado)
<mark> Texto marcado (highlight amarelo)
<small> Texto pequeno (letra miúda)

📏 4. Quebra de Linha e Linha Horizontal

quebras.html 
<!-- Quebra de linha (tag vazia, não fecha) -->
<p>Primeira linha<br>Segunda linha</p>

<!-- Linha horizontal (separador) -->
<p>Texto acima da linha</p>
<hr>
<p>Texto abaixo da linha</p>

📌 Tags Vazias

<br> e <hr> são tags vazias (não têm conteúdo), então não precisam de tag de fechamento.

🎨 5. Citações e Código

citacoes.html 
<!-- Citação curta (inline) -->
<p>O professor disse: <q>Estudem todos os dias</q> para passar.</p>

<!-- Citação longa (bloco) -->
<blockquote>
    Este é um trecho longo de um livro.
    Aparece recuado automaticamente.
</blockquote>

<!-- Código inline (na mesma linha) -->
<p>Use a tag <code>para mostrar código</code> na texto.</p>

<!-- Bloco de código -->
<pre>
    Este texto mantém
    a formatação original
    (espaços e quebras)
</pre>

✍️ Exercício Prático

📝 Criando uma Página de Biografia

Crie um arquivo biografia.html com:

  1. Um <h1> com seu nome
  2. Um <h2> com seu apelido/cargo
  3. Três <p> sobre você
  4. Use <strong> para destacar palavras importantes
  5. Use <em> para enfatizar
  6. Use <br> para quebrar linhas
  7. Adicione uma <hr> no final

📝 Código de Exemplo Completo

exercicio-completo.html 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Biografia</title>
</head>
<body>
    <h1>Davyd Lucas</h1>
    <h2>Desenvolvedor Front-End em Aprendizado</h2>
    
    <p>
        Olá! Sou o <strong>Davyd</strong>, 
        estou aprendendo <em>desenvolvimento web</em>.
    </p>
    
    <p>
        Minha jornada começou em 2024. 
        Estudo <strong>HTML</strong>, 
        <strong>CSS</strong> e 
        <strong>JavaScript</strong>.
    </p>
    
    <p>
        Meu objetivo é trabalhar com 
        <strong>front-end</strong><br>
        em uma boa empresa.</p>
    
    <hr>
    <p><small>Criado em 2026</small></p>
</body>
</html>

🎯 Resumo da Aula

Tags Aprendidas:

  • <h1> a <h6> - Títulos
  • <p> - Parágrafos
  • <strong> - Negrito (importante)
  • <em> - Itálico (ênfase)
  • <br> - Quebra de linha
  • <hr> - Linha horizontal
  • <blockquote> - Citação longa
  • <code> - Código inline
  • <pre> - Bloco de código formatado
