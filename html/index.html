📖 HTML: A Base da Web

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem que define a estrutura de páginas web [web:1].

exemplo.html 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro site!</p>
</body>
</html>

🏗️ Como Funciona a Web

🔍 Você sabia?

Quando você acessa um site, seu navegador (Chrome, Firefox) pede o arquivo HTML para um servidor. O servidor envia o HTML, e o navegador "desenha" a página na sua tela!

Para começar a programar em HTML, você precisa apenas de:
