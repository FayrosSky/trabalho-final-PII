📖 O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). É a linguagem que define a apresentação de documentos HTML [web:2].

💡 Analogia: HTML + CSS = Casa Completa

Se HTML é a estrutura da casa (paredes, portas, telhado), então CSS é a decoração (pintura, móveis, estilo):

  • HTML = Paredes brutas de cimento
  • CSS = Tinta colorida, papel de parede, iluminação

🎯 Sintaxe Básica do CSS

Um regra CSS tem três partes:

sintaxe-basica.css 
/* SELETOR { PROPIEDADE: VALOR; } */

h1 /* seletor */ {
    color: red;        /* propriedade: valor */
    font-size: 20px;   /* propriedade: valor */
    background: #fff;/* propriedade: valor */
}
Seletor Elemento HTML que vai receber o estilo (ex: h1, p, .classe)
Propriedade O que você quer mudar (ex: color, font-size, margin)
Valor O valor da propriedade (ex: red, 20px, #fff)

📝 3 Formas de Aplicar CSS

1. CSS Inline (dentro do HTML)

inline.html 
<p style="color: blue; font-size: 18px;">
    Este texto está azul e maior
</p>

⚠️ Não recomendado! Mistura conteúdo com estilo. Use apenas para testes rápidos.

2. CSS Interno (na mesma página)

interno.html 
<!DOCTYPE html>
<html>
<head>
    <style>
        h1 {
            color: green;
        }
    </style>
</head>
<body>
    <h1>Título verde</h1>
</body>
</html>

✅ Bom para páginas únicas. Mas o ideal é usar CSS externo.

3. CSS Externo (arquivo separado) - MELHOR OPÇÃO

externo.html 
<!-- NO HTML (index.html) -->
<head>
    <link 
        rel="stylesheet" 
        href="style.css"
    >
</head>

<!-- NO ARQUIVO (style.css) -->
p {
    color: purple;
    font-size: 16px;
}

✅ RECOMENDADO! Separa conteúdo (HTML) de estilo (CSS). Reutilizável em várias páginas.

🎨 Propriedades CSS Mais Comuns

Cores e Textos

cores-texto.css 
p {
    color: #6366f1;           /* cor do texto (hex) */
    font-size: 16px;          /* tamanho da fonte */
    font-weight: bold;        /* negrito */
    text-align: center;      /* alinha texto: center/left/right */
    text-decoration: none;   /* remove sublinhado */
    line-height: 1.6;        /* espaçamento entre linhas */
}

Box Model (Margens e Paddings)

box-model.css 
div {
    margin: 20px;             /* espaço FORA do elemento */
    padding: 15px;            /* espaço DENTRO do elemento */
    border: 2px solid black; /* borda: largura tipo cor */
    width: 300px;             /* largura */
    height: 200px;            /* altura */
}

📌 Box Model Explicado

Todo elemento HTML é uma "caixa":

content → padding → border → margin

✍️ Exercício Prático: Primeiro Estilo

📝 Criando seu Primeiro CSS

  1. Crie um arquivo index.html com:
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="style.css">
    <title>Meu Primeiro CSS</title>
</head>
<body>
    <h1>Olá, CSS!</h1>
    <p>Este é meu primeiro estilo.</p>
</body>
</html>
  1. Crie um arquivo style.css na mesma pasta:
body {
    font-family: Arial, sans-serif;
    background: #f0f0f0;
    padding: 50px;
}

h1 {
    color: #6366f1;
    text-align: center;
}

p {
    color: #666;
    font-size: 18px;
    text-align: center;
}
  1. Abra index.html no navegador e veja o resultado! 🎉

🎯 Resumo da Aula

  • CSS = Cascading Style Sheets
  • CSS define a apresentação (estilo) do HTML
  • Sintaxe: seletor { propriedade: valor; }
  • 3 formas: inline (não use), interno, externo (ideal)
  • Propriedades comuns: color, font-size, margin, padding, border
  • Use arquivo .css separado para organizar seu código
