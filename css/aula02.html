Seletores CSS

📖 O que são Seletores?

Seletores são "endereços" que dizem ao CSS qual elemento HTML estilizar [web:2].

🎯 1. Seletor de Elemento (Tag)

seletor-elemento.css 
/* Seleciona TODOS os <p> da página */
p {
    color: blue;
    font-size: 16px;
}

🎯 2. Seletor de Classe (.classe)

A classe é reutilizável em vários elementos:

classe.html 
<!-- HTML -->
<p class="destaque">Texto importance</p>
<h1 class="destaque">Título importante</h1>
classe.css 
/* CSS - note o PONTO (.) antes do nome */
.destaque {
    color: #6366f1;
    font-weight: bold;
}

✅ Use classes! São reutilizáveis eorganizadas.

🎯 3. Seletor de ID (#id)

O ID é único - só pode usar uma vez na página:

id.html 
<!-- HTML -->
<header id=" topo"></header>
<footer id="footer"></footer>
id.css 
/* CSS - note o HASHTAG (#) antes do nome */
#topo {
    background: #333;
    color: white;
}
Classe (.classe) ID (#id)
Reutilizável em vários elementos Único (apenas 1 elemento)
Use para componentes Use para seções únicas
PREFERIDO Usar menos

🎯 4. Seletor Universal (*)

universal.css 
/* Seleciona TODOS os elementos */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

✅ Usado no CSS Reset para padronizar todos os navegadores.

🎯 5. Seletor de Atributo

atributo.css 
/* Seleciona inputs tipo email */
input[type="email"] {
    border: 2px solid blue;
}

/* Seleciona todos links externos */
a[href*="gmail.com"] {
    color: red;
}

🎯 6. Seletor Descendente (espaço)

descendente.css 
/* Seleciona p DENTRO de .container */
.container p {
    color: gray;
}

/* Seleciona li DENTRO de ul */
ul li {
    list-style: square;
}

🎯 7. Seletor Filho (>)

filho.css 
/* Seleciona SOMENTE filhos diretos */
.container > p {
    color: blue;
}

/* Diferença: */
.container p   /* todos os netos, bisnetos... */
.container > p /* apenas filhos diretos */

🎯 8. Seletor de Irmãos Adjacentes (+)

irmao.css 
/* Seleciona p IMEDIATAMENTE após h1 */
h1 + p {
    margin-top: 0;
}

🎯 9. Pseudo-classes (:hover, :focus, etc)

pseudo-classes.css 
/* Quando passa o mouse */
a:hover {
    color: red;
    text-decoration: underline;
}

/* Quando clica no input */
input:focus {
    border-color: blue;
    outline: none;
}

/* Primeiro elemento */
li:first-child {
    font-weight: bold;
}

/* Par (2, 4, 6...) */
tr:nth-child(even) {
    background: #f0f0f0;
}

✍️ Exercício Prático: Estilize um Card

📝 Criando um Card com Seletores

card.html 
<div class="card">
    <h2 class="card-title">Título</h2>
    <p class="card-text">Texto</p>
    <a href="#" class="btn">Link</a>
</div>
card.css 
.card {
    border: 1px solid #ddd;
    padding: 20px;
    border-radius: 8px;
}

.card-title {
    color: #6366f1;
    font-size: 24px;
}

.btn:hover {
    background: #6366f1;
    color: white;
}

🎯 Resumo da Aula

Seletores Aprendidos:

  • elemento - Seleciona tag (ex: p, h1)
  • .classe - Seleciona classe (reutilizável)
  • #id - Seleciona ID (único)
  • * - Seleciona todos
  • [attr] - Seleciona atributo
  • div p - Descendente (todos os filhos)
  • div > p - Filho direto
  • h1 + p - Irmão adjacente
  • a:hover - Pseudo-classe
