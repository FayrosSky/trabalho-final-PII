Flexbox - Layout Flexível

📖 O que é Flexbox?

Flexbox é um modelo de layout do CSS que organiza elementos em uma direção (horizontal ou vertical) de forma inteligente e responsiva.

💡 Por que Flexbox?

  • ✅ Alinhamento fácil (centralizar é trivial!)
  • ✅ Responsivo por natureza
  • ✅ Distribui espaço automaticamente
  • ✅ Substitui floats e hacks antigos

🎯 1. Ativando Flexbox

Aplicar display: flex no pai torna os filhos "flexíveis":

exemplo-basico.html 
<div class="container">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>
exemplo-basico.css 
.container {
    display: flex;  /* ativa flexbox */
}

.item {
    width: 100px;
    height: 100px;
    background: #6366f1;
    color: white;
    text-align: center;
    line-height: 100px;
    font-size: 24px;
}

Conceito importante: Quando você aplica display: flex no pai, os filhos automaticamente ficam lado a lado (horizontalmente).

🎯 2. Eixo Principal vs Eixo Cruzado

Flexbox trabalha com dois eixos:

Eixo Descrição Padrão
Principal Direção dos itens Horizontal (←→)
Cruzado Perpendicular ao principal Vertical (↑↓)

🎯 3. flex-direction - Direção dos Itens

Define a direção do eixo principal:

flex-direction.css 
/* Horizontal da esquerda para direita (PADRÃO) */
.container {
    flex-direction: row;
}

/* Horizontal da direita para esquerda */
.container {
    flex-direction: row-reverse;
}

/* Vertical de cima para baixo */
.container {
    flex-direction: column;
}

/* Vertical de baixo para cima */
.container {
    flex-direction: column-reverse;
}

💡 Dica: Use column para listas verticais ou layouts de coluna.

🎯 4. justify-content - Alinhamento no Eixo Principal

Controla como os itens se distribuem horizontalmente:

justify-content.css 
/* À esquerda (PADRÃO) */
.container {
    justify-content: flex-start;
}

/* Centralizado */
.container {
    justify-content: center;
}

/* À direita */
.container {
    justify-content: flex-end;
}

/* Espaçados igualmente (começa/termina nas bordas) */
.container {
    justify-content: space-between;
}

/* Espaçados igualmente (mesmo espaço nas bordas) */
.container {
    justify-content: space-around;
}

/* Espaçados igualmente (mesmo espaço em todos) */
.container {
    justify-content: space-evenly;
}
Valor Resultado Visual
flex-start ■ ■ ■ _______
center _ _ ■ ■ ■ _ _
flex-end _______ ■ ■ ■
space-between ■ _______ ■ _______ ■
space-around _ ■ _ _ ■ _ _ ■ _

🎯 5. align-items - Alinhamento no Eixo Cruzado

Controla alinhamento vertical (quando flex-direction é row):

align-items.css 
/* Estica para preencher (PADRÃO) */
.container {
    align-items: stretch;
}

/* Topo */
.container {
    align-items: flex-start;
}

/* Centro */
.container {
    align-items: center;
}

/* Fundo */
.container {
    align-items: flex-end;
}

/* Linha de base (para textos) */
.container {
    align-items: baseline;
}

⭐ Caso de Uso Comum: Centralizar Everything

Para centralizar um elemento perfeitamente:

.container {
    display: flex;
    justify-content: center;    /* horizontal */
    align-items: center;        /* vertical */
    height: 100vh;              /* altura total da tela */
}

🎯 6. flex-wrap - Quebra de Linha

Define se os itens devem quebrar para outra linha:

flex-wrap.css 
/* Tudo numa linha (estica) - PADRÃO */
.container {
    flex-wrap: nowrap;
}

/* Quebra quando não cabe */
.container {
    flex-wrap: wrap;
}

/* Quebra na direção inversa */
.container {
    flex-wrap: wrap-reverse;
}

💡 Use wrap para criar layouts responsivos que se adaptam a telas menores.

🎯 7. gap - Espaçamento entre Itens

Adiciona espaço entre os itens (muito útil!):

gap.css 
/* Espaço horizontal e vertical */
.container {
    gap: 20px;
}

/* Espaço separado */
.container {
    gap: 10px 20px;  /* row gap | column gap */
}

🎯 8. Propriedades nos ITENS (filhos)

flex - Controlando o Tamanho dos Itens

flex-item.css 
.item {
    flex: 1;  /* ocupa espaço proporcional */
}

/* Shorthand: flex-grow | flex-shrink | flex-basis */
.item {
    flex: 1 1 200px;
}

/* grow: quanto cresce */
/* shrink: quanto encolhe */
/* basis: tamanho inicial */
exemplo-grow.html 
<div class="container">
    <div class="item">1</div>
    <div class="item" style="flex: 2;">2 (duplo)</div>
    <div class="item">3</div>
</div>

order - Mudar Ordem Visual

order.css 
.item1 { order: 2; }
.item2 { order: 1; }
.item3 { order: 3; }

/* Visual: item2, item1, item3 */

align-self - Alinhamento Individual

align-self.css 
/* Um item diferente dos outros */
.item2 {
    align-self: flex-end;  /* vai para o fundo */
}

✍️ Exercício Prático: Menu de Navegação

📝 Criando um Menu com Flexbox

Crie um menu de navegação responsivo:

menu.html 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Menu Flexbox</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <nav class="navbar">
        <div class="logo">Logo</div>
        <ul class="menu">
            <li><a href="#">Início</a></li>
            <li><a href="#">Sobre</a></li>
            <li><a href="#">Contato</a></li>
        </ul>
    </nav>
</body>
</html>
style.css 
*: {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem 2rem;
    background: #333;
}

.logo {
    color: white;
    font-size: 1.5rem;
    font-weight: bold;
}

.menu {
    display: flex;
    list-style: none;
    gap: 2rem;
}

.menu a {
    color: white;
    text-decoration: none;
    transition: color 0.3s;
}

.menu a:hover {
    color: #6366f1;
}

🎯 Resumo da Aula

Propriedades do PAI (container):

  • display: flex - ativa flexbox
  • flex-direction - direção (row/column)
  • justify-content - alinhamento horizontal
  • align-items - alinhamento vertical
  • flex-wrap - se quebra linha
  • gap - espaçamento entre itens

Propriedades dos ITENS (filhos):

  • flex - controla crescimento/encolhimento
  • order - muda ordem visual
  • align-self - alinhamento individual
