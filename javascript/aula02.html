Variáveis e Tipos de Dados

📖 O que é uma Variável?

Variável é um "recipiente" que armazena dados na memória do computador. Você pode guardar números, textos, e outros valores.

💡 Analogia: Variável = Caixa com Etiqueta

Imagine uma caixa com um rótulo. Dentro da caixa tem um valor, e você acessa o valor pelo nome da caixa (variável).

🎯 3 Formas de Declarar Variáveis

1. var (ANTIGO - NÃO USE)

var-antigo.js 
var nome = "Davyd"; /* antigo, não use mais */

2. let (variável que MUDA) ✅

let.js 
let idade = 17;
idade = 18;  /* pode mudar o valor */

console.log(idade);  /* 18 */

✅ Use let quando o valor for mudar no futuro.

3. const (constante que NÃO MUDA) ✅ RECOMENDADO

const.js 
const pi = 3.14159;
pi = 3.14;  /* ERRO! Não pode mudar const */

const nome = "CodeAprende";
console.log(nome);  /* CodeAprende */

✅ Use const por padrão! Só use let se precisar mudar o valor.

📊 Tipos de Dados em JavaScript

1. String (texto)

string.js 
const nome = "Davyd";
const frase = 'Olá, mundo!';
const texto = `Template string`;

console.log("Nome: " + nome);  /* concatenação */
console.log(`Nome: ${nome}`);  /* template string (melhor!) */

2. Number (número)

number.js 
const idade = 17;           /* inteiro */
const preco = 9.99;       /* decimal */
const negativo = -5;

console.log(idade + 3);   /* 20 */
console.log(preco * 2);   /* 19.98 */

3. Boolean (verdadeiro/falso)

boolean.js 
const estáLogado = true;
const éAdulto = false;
const maiorIdade = idade >= 18;  /* false */

4. Undefined (não definido)

undefined.js 
let variavel;
console.log(variavel);  /* undefined */

const nome = "Davyd";
nome = undefined;  /* ERRO! Não pode mudar const */

5. Null (vazio/intencional)

null.js 
const usuario = null;  /* usuário não existe */

if (usuario === null) {
    console.log("Nenhum usuário selecionado");
}
Tipo Exemplo Descrição
String "texto" Texto entre aspas
Number 42, 3.14 Números (inteiro ou decimal)
Boolean true, false Verdadeiro ou falso
Undefined let x; Declaramento sem valor
Null null Valor vazio intencional

🎯 Operadores em JavaScript

Operadores Matemáticos

operadores-matematicos.js 
let a = 10;
let b = 3;

console.log(a + b);   /* 13 - adição */
console.log(a - b);   /* 7 - subtração */
console.log(a * b);   /* 30 - multiplicação */
console.log(a / b);   /* 3.33 - divisão */
console.log(a % b);   /* 1 - resto da divisão */
console.log(a ** b);  /* 1000 - potência */

Operadores de Comparação

operadores-comparacao.js 
const a = 5;
const b = "5";

console.log(a == b);   /* true - igual (valor) */
console.log(a === b);  /* false - estrito (valor e tipo) */
console.log(a != b);   /* false - diferente */
console.log(a !== b);  /* true - estritamente diferente */
console.log(a > b);    /* false - maior */
console.log(a < b);    /* false - menor */
console.log(a >= b);   /* true - maior ou igual */

⚠️ == vs ===

Sempre use === (estrito)! == faz conversão automática de tipo, o que pode causar bugs.

Operadores Lógicos

operadores-logicos.js 
const a = true;
const b = false;

console.log(a && b);   /* false - AND (e) */
console.log(a || b);   /* true - OR (ou) */
console.log(!a);       /* false - NOT (negação) */

// Usando em condições
const idade = 20;
const temPrivilegio = idade >= 18 && true;
console.log(temPrivilegio);  /* true */

✍️ Exercício Prático: Calculadora de IMC

📝 Criando uma calculadora simples

// Calculadora de IMC (Índice de Massa Corporal)

const peso = 70;  // em kg
const altura = 1.75;  // em metros

// Fórmula: IMC = peso / (altura * altura)
const imc = peso / (altura * altura);

console.log(`IMC: ${imc.toFixed(2)} `);  /* 22.86 */

// Verificar categoria
let categoria;

if (imc < 18.5) {
    categoria = "Abaixo do peso";
} else if (imc < 25) {
    categoria = "Peso normal";
} else if (imc < 30) {
    categoria = "Sobrepeso";
} else {
    categoria = "Obesidade";
}

console.log(`Categoria: ${categoria}`);  /* Peso normal */

🎯 Resumo da Aula

  • let = variável que muda
  • const = constante (não muda) - USE POR PADRÃO!
  • String = texto ("abc")
  • Number = número (42, 3.14)
  • Boolean = verdadeiro/falso (true/false)
  • === > sempre use estrito (não use ==)
  • && = AND, || = OR, ! = NOT
