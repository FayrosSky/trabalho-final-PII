Funções em JavaScript

📖 O que é uma Função?

Função é um bloco de código que realiza uma tarefa específica. Você pode "chamar" a função quantas vezes quiser, sem precisar reescrever o código.

💡 Analogia: Função = Receita de Bolo

Uma receita tem passos para fazer um bolo. Você pode usar a mesma receita várias vezes para fazer vários bolos. A função é igual: uma "receita" de código que você reutiliza.

🎯 1. Função Declarativa (Tradicional)

funcao-tradicional.js 
/* Sintaxe: function nome() { } */

function saudacao() {
    console.log("Olá, mundo!");
}

// Chamando a função
saudacao();  /* Saída: Olá, mundo! */

// Pode chamar várias vezes
saudacao();  /* Olá, mundo! */
saudacao();  /* Olá, mundo! */

🎯 2. Função com Parâmetros

Parâmetros são "variáveis" que a função recebe:

funcao-parametros.js 
function saudacaoNome(nome) {
    console.log("Olá, " + nome + "!");
}

saudacaoNome("Davyd");     /* Olá, Davyd! */
saudacaoNome("Maria");     /* Olá, Maria! */

// Múltiplos parâmetros
function somar(a, b) {
    console.log(a + b);
}

somar(5, 3);    /* 8 */
somar(10, 20);   /* 30 */

💡 Parâmetros são como variáveis locais da função.

🎯 3. Função com Return (Retorno)

return devolve um valor para quem chamou a função:

funcao-return.js 
function somar(a, b) {
    return a + b;  /* devolve o resultado */
}

const resultado = somar(5, 3);
console.log(resultado);  /* 8 */

// Usando o retorno
const dobro = somar(5, 5) * 2;
console.log(dobro);  /* 20 */

// O return termina a função
function verificarIdade(idade) {
    if (idade >= 18) {
        return "Maior de idade";
    }
    return "Menor de idade";  /* só chega aqui se idade < 18 */
}

console.log(verificarIdade(20));  /* Maior de idade */
console.log(verificarIdade(15));  /* Menor de idade */

⚠️ Importante sobre return

Quando o return é executado, a função PARALIZA imediatamente. Qualquer código depois do return não é executado.

🎯 4. Função com Parâmetro Padrão

parametro-padrao.js 
function saudacao(nome = "Visitante") {
    console.log(`Olá, ${nome}!`);
}

saudacao();           /* Olá, Visitante! */
saudacao("Davyd");    /* Olá, Davyd! */

// Valores padrão para números
function multiplicar(a, b = 2) {
    return a * b;
}

console.log(multiplicar(5));      /* 10 (5 * 2) */
console.log(multiplicar(5, 3));   /* 15 (5 * 3) */

🎯 5. Arrow Function (ES6+) ⭐ RECOMENDADO

Sintaxe mais curta e moderna para criar funções:

arrow-function.js 
/* Sintaxe: const nome = (param) => { } */

// Função tradicional
function saudacao() {
    console.log("Olá!");
}

// Arrow function (mesmo comportamento)
const saudacao = () => {
    console.log("Olá!");
};

saudacao();  /* Olá! */

Arrow Function com Parâmetros

arrow-parametros.js 
// Um parâmetro (pode tirar parênteses)
const dobrar = num => num * 2;

console.log(dobrar(5));  /* 10 */

// Múltiplos parâmetros (precisa de parênteses)
const somar = (a, b) => a + b;

console.log(somar(3, 7));  /* 10 */

// Sem parâmetro
const saudacao = () => {
    console.log("Olá!");
};

Arrow Function com Return Implícito

arrow-return-implicito.js 
// Traditional com return
function quadrado(num) {
    return num * num;
}

// Arrow function com return implícito
const quadrado = num => num * num;

// Mesma coisa! O return é automático quando não tem { }

console.log(quadrado(4));  /* 16 */

📌 Quando usar Arrow Function?

  • ✅ Para funções curtas e simples
  • ✅ Como callback em métodos de array (map, filter, reduce)
  • ✅ Quando não precisa de this
  • ⚠️ Não use para métodos de objeto

🎯 6. Função IIFE (Imediatamente Executada)

iife.js 
/* Função que executa assim que é definida */

(function() {
    console.log("Executou imediatamente!");
})();

// Arrow function IIFE
(() => {
    console.log("Também executa agora!");
})();

✍️ Exercício Prático: Calculadora Completa

📝 Criando uma calculadora com funções

// Calculadora com funções

const somar = (a, b) => a + b;
const subtrair = (a, b) => a - b;
const multiplicar = (a, b) => a * b;
const dividir = (a, b) => {
    if (b === 0) {
        return "Erro: divisão por zero!";
    }
    return a / b;
};

const porcentagem = (valor, porcento) => {
    return (valor * porcento) / 100;
};

// Testando
console.log(`5 + 3 = ${somar(5, 3)}`);        /* 8 */
console.log(`10 - 4 = ${subtrair(10, 4)}`);     /* 6 */
console.log(`6 * 7 = ${multiplicar(6, 7)}`);    /* 42 */
console.log(`20 / 4 = ${dividir(20, 4)}`);      /* 5 */
console.log(`10% de 200 = ${porcentagem(200, 10)}`);  /* 20 */

// Calculadora interativa
const calculadora = (a, b, operacao) => {
    return operacao(a, b);
};

const resultado = calculadora(10, 5, somar);
console.log(resultado);  /* 15 */

🎯 Resumo da Aula

  • Função = bloco de código reutilizável
  • function nome() {} = função tradicional
  • () => {} = arrow function (moderna)
  • Parâmetros = valores que a função recebe
  • return = devolve valor e para a função
  • Parâmetro padrão = (x = valor)
  • Return implícito = (x) => x * 2 (sem { })
  • Use arrow functions para código mais limpo!
