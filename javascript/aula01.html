📖 O que é JavaScript?

JavaScript (JS) é uma linguagem de programação que torna sites interativos e dinâmicos. É uma das três tecnologias fundamentais da web, junto com HTML e CSS.

💡 Analogia: HTML + CSS + JavaScript = Casa Inteligente

  • HTML = Estrutura (paredes, portas)
  • CSS = Decoração (pintura, estilo)
  • JavaScript = Funcionalidades (luzes, portão automático)

🎯 O que JavaScript pode fazer?

  • 🖱️ Responder a cliques e eventos
  • 🔄 Atualizar conteúdo sem recarregar
  • 📡 Conversar com servidor (APIs)
  • 🎬 Criar animações
  • ✅ Validar formulários
  • 🍪 Armazenar dados (LocalStorage)

📝 3 Formas de Adicionar JavaScript

1. JavaScript Inline (NÃO USE)

inline.html 
<button onclick="alert('Olá!')">Clique</button>

2. JavaScript Interno (para testes)

interno.html 
<script>
    alert("Olá, Mundo!");
</script>

3. JavaScript Externo (MELHOR OPÇÃO) ✅

externo.html 
<!-- NO HTML -->
<script src="script.js"></script>

<!-- NO ARQUIVO script.js -->
alert("Olá, Mundo!");

✍️ Exercício Prático: Seu Primeiro JavaScript

📝 Criando um arquivo JS

  1. Crie um arquivo index.html
  2. Crie um arquivo script.js na mesma pasta
  3. Conecte eles no HTML
<!-- index.html -->
<!DOCTYPE html>
<html>
<body>
    <h1 id="titulo">Olá!</h1>
    <button id="botao">Clique</button>
    
    <script src="script.js"></script>
</body>
</html>
// script.js
const botao = document.getElementById("botao");

botao.addEventListener("click", function() {
    alert("Botão clicado!");
    document.getElementById("titulo").innerText = "Clicou!!!";
    document.getElementById("titulo").style.color = "red";
});

🎯 Resumo da Aula

  • JavaScript = linguagem de programação interativa
  • HTML = estrutura, CSS = estilo, JS = interatividade
  • Use arquivo .js externo
  • console.log() para depurar
  • document.getElementById() para selecionar
  • addEventListener() para eventos
